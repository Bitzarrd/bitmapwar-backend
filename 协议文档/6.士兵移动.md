6.士兵移动.md
# 方向：Server-->Client
## 字段
| 字段名 | 字段类型 | 字段说明 |
|-------|-------|-------|
| method  | string  | 固定为'Update'  |
| payload  | Cell[]  | 变更的地块信息  |
| turn  | int  | 当前走到第几步  |
| statistics  | Statistics  | 统计信息  |
| total_bonus | String  | 当前轮的总奖励  |
| jackpot | String  | 累计奖池  |
| dead_cells | DeadCenterCell | 战死的士兵的中心  |
| action_log | ActionLog or Null | 定义一个战报 |
