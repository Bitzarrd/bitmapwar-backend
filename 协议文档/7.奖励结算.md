# 方向：Server-->Client
## 字段
| 字段名 | 字段类型 | 字段说明 |
|-------|-------|-------|
| method  | string  | 固定为'Settlement'  |
| my_statistics  | Statistics  | 我的统计信息  |
| statistics  | Statistics[]  | 4个阵营的统计信息  |
| next_round  | int  | 下一回合开始时间  |
| user  | User  | 用户信息  |
| earning  | string  | 收益  |
| rank  | User[]  | 用户排名  |
